# Analog Clock With Numbers

## Introduction

Analog Clock With Numbers is a Java application that displays an analog clock with numerical hour markers. It provides a visual representation of the current time and serves as a simple, user-friendly clock display.

## Author

- Author: Priyanshu Chaudhary
- GitHub: [@whitedevil94](https://github.com/whitedevil94)

## Features

- Displays an analog clock with dark blue clock face.
- Numerical hour markers are placed inside the clock face.
- Clock hands indicate the current time in yellow.
- A digital time display in black shows the current time and day name.

## How to Run

1. Compile the Java source code.
2. Run the program, and an analog clock window will appear.
3. The clock will automatically update every second to show the current time.

## Color Scheme

- Dark blue is used for the clock face.
- Yellow is used for the numerical hour markers and clock hands.
- Black is used for the digital time display outside the clock face.
